# CNE Portfolio Website

This project is a personal portfolio website for Dmitriy Kerget, focused on cloud network engineering experience, projects, and contact information.

## What This Project Is About

- Presents a professional profile and career background
- Highlights project work and technical focus areas
- Provides contact information and a downloadable resume
- Serves as a static website suitable for GitHub Pages hosting

## Tech Stack / Languages

- HTML5
- CSS3

## Project Structure

- `index.html` - Main landing page
- `about.html` - About page
- `projects.html` - Projects page
- `contact.html` - Contact page
- `assets/css/style.css` - Shared stylesheet
- `assets/images/` - Image assets
- `assets/docs/resume.pdf` - Resume file
